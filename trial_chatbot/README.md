## Quick Start

### Option 1: Docker (Recommended)
```bash
cp .env.example .env
# Edit .env with your LLM endpoint
docker compose up --build
```

### Option 2: Local Python
```bash
pip install -r requirements.txt
cp .env.example .env
# Edit .env with your LLM endpoint
python app.py
```

### Chat API
```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello!"}'
```
